# Code Repositories

## 1. SVCCA (Google Research)
- **Source**: `https://github.com/google/svcca`
- **Location**: `code/svcca/`
- **Purpose**: Reference implementation of SVCCA and PWCCA.

## 2. CKA (Custom Implementation)
- **File**: `cka.py`
- **Purpose**: Provides a clean PyTorch implementation of Linear CKA for use in experiments.

## Recommended Usage
Use `cka.py` to compare hidden state matrices $X$ and $Y$ of size $(N, D1)$ and $(N, D2)$.
